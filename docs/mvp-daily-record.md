# 数字人日记 MVP 功能说明书

## 目标用户访谈摘要
- **核心画像**：25-35 岁的一线城市职场人士，关注自我成长与情绪管理，使用手机端数字人应用记录生活。
- **访谈对象**：
  1. **Joyce（产品经理，29 岁）**：习惯睡前回顾一天，苦于文字输入耗时，希望语音快速记录并生成结构化总结。
  2. **Leo（设计师，31 岁）**：常在通勤时记录灵感，担心数据散落；希望 AI 能帮忙整理、回放重点。
  3. **Mia（新晋妈妈，27 岁）**：想捕捉育儿瞬间，偏好图像+语音混合记录，并希望能与家人分享精选片段。
- **关键需求洞察**：
  - 记录必须“随手”且多模态（语音、文字、图片）。
  - 需要自动化梳理与回顾，提供结构化总结与情绪洞察。
  - 期待数字人以具象化形象进行每日复盘交流，提高陪伴感。

## 每日记录用户旅程图
| 阶段 | 接触点 | 用户行为 | 用户想法/情绪 | 痛点机会 |
| --- | --- | --- | --- | --- |
| 触发 | 手机通知、可穿戴设备提醒 | 接收数字人提醒，决定记录时间点（晨间、午间、睡前） | “我有没有遗漏重要事情？” | 提供智能提醒策略，基于日程或情绪波动触发 |
| 记录 | App 首页、语音输入、快捷卡片 | 通过语音/文字/图片快速记录事件、情绪、待办 | “希望快一点，不想打字” | 语音实时转文字、多模态上传、模板提示 |
| 自动整理 | 云端 AI、标签系统 | AI 自动抽取关键词、情绪标签、待办清单 | “别让我再手动整理了” | 自动分类、结构化卡片、情绪检测 |
| 回顾 | App 日总结页、数字人对话 | 浏览总结、与数字人对话复盘、收到行为建议 | “我今天表现如何？” | 可视化时间轴、情绪波动图、对话式建议 |
| 扩散 | 分享面板、家人空间 | 选择性分享音频/视频摘要给家人或私密群 | “我希望被理解但不想泄露隐私” | 精选片段生成与隐私控制 |

## 功能分阶段规划
| 阶段 | 功能项 | 类型 |
| --- | --- | --- |
| 数据采集 | 语音速记（转写+噪声过滤） | 必须 |
| 数据采集 | 文字速记（模板填空、快捷标签） | 必须 |
| 数据采集 | 图片上传（自动打标签） | 可选 |
| 数据采集 | 智能提醒（基于时间/位置） | 可选 |
| 内容回顾 | 每日摘要（事件+情绪+待办） | 必须 |
| 内容回顾 | 可视化时间轴与情绪曲线 | 可选 |
| 内容回顾 | 可搜索的历史档案（关键词/情绪） | 可选 |
| 数字人展示 | 数字人对话式复盘（语音 + 文本） | 必须 |
| 数字人展示 | 可配置数字人形象（头像、语音） | 可选 |
| 数字人展示 | 分享模式（生成可分享短视频/卡片） | 可选 |

## MVP 核心流程
1. **提醒触达**：智能推送提醒（晨/晚两个默认时段，可手动触发）。
2. **快速记录**：用户在 App 首页使用语音按钮或文字卡片录入信息，可上传图片辅助。
3. **AI 处理**：后端完成语音转写、情绪分析、事件抽取并生成待办/亮点。
4. **数字人复盘**：数字人以对话形式向用户呈现摘要，并允许追问细节或追加记录。
5. **每日归档**：生成可浏览的日总结卡片，历史可按日期检索。

## MVP 界面草图（文字描述）
- **首页/记录面板**：
  - 顶部：今日问候 + 智能提醒卡片。
  - 中部：大号语音按钮（主 CTA），旁边为文字速记按钮与图片上传入口。
  - 底部：最近三条记录列表，支持继续编辑。
- **AI 摘要页**：
  - 上方：数字人 3D 头像 + 今日情绪指示灯（颜色）。
  - 中间：卡片式展示“今日亮点 / 情绪分析 / 待办提醒”。
  - 下方：对话输入框，可语音或文字追问。
- **历史归档页**：
  - 日期时间轴，点击展开当日摘要。
  - 搜索栏支持关键词/情绪筛选。

## 优先级矩阵
| 优先级 | 功能 | 理由 |
| --- | --- | --- |
| P0 | 语音/文字记录、AI 摘要、数字人对话复盘 | 构成最小闭环（记录→总结→回顾）|
| P1 | 图片上传、待办同步提醒、历史搜索 | 提升记录丰富度与可追溯性 |
| P2 | 可视化情绪曲线、数字人外观自定义、分享模式 | 提升体验与传播性 |
